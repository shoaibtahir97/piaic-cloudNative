Document Object Model

JavaScript is used on the browser. HTML is loaded on the window/DOM.
If you want to access/target something on the HTML you have to use DOM
The HTML is converted into a document, which has different objects through which we can access elements and perform multiple functions

  • QuerySelector: It is used to target any element in the HTML document.
    You can pass tag name, id and class. If there are multiple elements with
    the same name then it will return the first element with that name, id or class.
  • QuerySelectorAll: It is used to target all the element in the HTML document.
    It returns an array
  • AddEventListener: It is another method of document which takes two parameters.
    First is the event which could be click, change, DOMContentLoaded
    Second parameter is a function which is called when the event is fired
    When using DOMContentLoaded event you just have to pass the function name. No need to call the function
    Because JavaScript itself calls the function when the content is loaded on the browser.
    When DOM content is loaded then the function is executed
  • element.classList.add('red'): Adds a value of red to the class attribute. element.classList.remove('red'): Removes the value of red from the class attribute
  • forEach: When we use QuerySelectorAll it returns the elements in an array and inside the array
    the method of AddEventListener i not available. To solve this issue we use forEach method to loop
    through the array generated by QuerySelectorAll and call the AddEventListener on each element.
