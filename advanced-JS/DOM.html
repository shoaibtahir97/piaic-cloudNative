Document Object Model

JavaScript is used on the browser. HTML is loaded on the window/DOM.
If you want to access/target something on the HTML you have to use DOM
The HTML is converted into a document, which has different objects through which we can access elements and perform multiple functions

  • QuerySelector: It is used to target any element in the HTML document.
    You can pass tag name, id and class. If there are multiple elements with
    the same name then it will return the first element with that name, id or class.
  • QuerySelectorAll: It is used to target all the element in the HTML document.
    It returns an array
